Giving them Life (Without JS)

Transitions
  • Occurs once
  • Dependent on element changes
  • Specify rules, but not amounts
  • Cannot control frame-by-frame
Animations
  • Can be repeated infinitely
  • Not tied to element changes
  • Specify rules and amounts they change
  • Can specify frames, percents, or timings

Finally, Transformations!

  • Using the rule: "transform"
  • Can be 2D or 3D
  • Types of Transformations
    • Rotations - one axis, or multiple
    • Translations - one axis (X, Y, Z), or multiple
    • Scaling - one axis, multiple, or all
  • Special Transformations
    • "perspective"
